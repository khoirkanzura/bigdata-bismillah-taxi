# Evaluasi Model Regresi untuk Estimasi Tarif Taksi Berbasis Spark dan HDFS

Repository ini berisi implementasi eksperimen Big Data untuk estimasi tarif taksi menggunakan dataset NYC Taxi Fare. Dataset diproses menggunakan Hadoop Distributed File System (HDFS), Apache Spark, PySpark, dan Spark ML Pipeline.

## Teknologi yang digunakan

- Apache Hadoop HDFS
- Apache Spark
- PySpark
- Spark ML Pipeline
- Python
- Matplotlib
- Pandas
